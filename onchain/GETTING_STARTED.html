⚡ Quick Start (5 Minutes)

✓ Good News: Everything is already set up! Just follow these steps to deploy.

1Update Environment

Edit the .env file with your Sepolia credentials:

PRIVATE_KEY=your_private_key_here SEPOLIA_RPC_URL=https://ethereum-sepolia-rpc.publicnode.com ETHERSCAN_API_KEY=your_etherscan_key_here # Optional

2Get Sepolia ETH

Visit a testnet faucet to get free ETH:

SepoliaFaucet.com Alchemy Faucet

3Verify Setup

Run these commands to ensure everything works:

npm run compile npm run test

Expected: All 78 tests pass in ~2 seconds

4Deploy to Sepolia

Run the deployment command:

npm run deploy:sepolia

Result: 3 contracts deployed with addresses logged

5Verify on Etherscan

Visit Etherscan Sepolia and search for your contract addresses:

Etherscan Sepolia
