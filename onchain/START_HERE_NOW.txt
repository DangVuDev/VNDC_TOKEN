TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

  Problem: "Error: insufficient funds"
  Solution: You need more Sepolia ETH. Get more from the faucet above.
  
  Problem: "Invalid private key format"
  Solution: Make sure it starts with 0x and is 66 characters long (0x + 64 hex).
  
  Problem: Tests fail
  Solution: Run "npm run compile" first, then "npm test"
  
  Problem: Deploy takes very long
  Solution: This is normal. Testnet can be slow. Give it 5-10 minutes.
